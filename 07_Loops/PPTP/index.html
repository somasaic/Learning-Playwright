56 - For Loop Basics

56_For_Loops.js For Loop

The basic for loop syntax: for(init; condition; update). Both pre-increment and post-increment work in the update expression.

// for(I;C;U)
for (let i = 0; i < 5; ++i) {
    console.log(i);
}

for (let i = 0; i < 5; i++) {
    console.log(i);
}

59 - While Loop

59_While_Loop.js While Loop

The while loop checks the condition before each iteration. It requires separate initialization, condition, and update (I, C, U).

// While Loop - Sister (For)
// I,C,u

let attempts = 0; // Init

while (attempts < 3) {
    console.log("Attempt", attempts);
    attempts++;
}

61 - Do-While Loop

61_DO_while.js Do-While

The do-while loop always executes the body at least once before checking the condition. This is the key difference from the while loop.

// let retry = 0;

// do {
//     console.log("Execute a code!");
//     console.log("Retrying.....", retry);
//     retry++;


// } while (retry < 3);


let a = 10;
// while (a < 10) {
//     console.log(a);
//     a++
// }



do {
    console.log(a);
    a++;
} while (a < 10);
